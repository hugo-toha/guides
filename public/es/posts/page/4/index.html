Cómo traducir los datos de la página de inicio

Este tema tiene soporte integrado para varios idiomas. Añade el idioma en el sitio Añadiendo el idioma a su sitio traducirá la interfaz de usuario a ese idioma (por ejemplo, los botones, las barras de navegación, etc.). Obtener el código del idioma Para traducir su sitio, necesitará el código de su idioma.
